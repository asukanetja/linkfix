LinkFix

LinkFix is a free Windows utility that lists and batch-modifies shortcut (.lnk) target paths.

Features

  • Search all .lnk files in a folder (including subfolders)
  • Display shortcut targets in a list
  • Batch replace specific text in shortcut paths
  • Export results as CSV
  • No installation required – just run the EXE
