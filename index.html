LinkFix

LinkFix is a free tool for Windows that lets you list and batch-replace shortcut (.lnk) targets.

Features

  • Search all .lnk files in a folder (including subfolders)
  • Display shortcut targets in a list
  • Batch replace specific strings in shortcut paths
  • Export results as CSV
